prerequisite) Build all build specification for Source\Import-Export_Proj.lvproj

1) Close Veristand if originally opened
2) Copy the content of VS Test Project in the <VS Public Directory>Projects folder
3) Open Import-Export_Proj.lvproj
4) (optional) if working with versions other than 2013, open the build specification, modify the Destination directory to current version and rebuild
5) Run Veristand Launcher Utility.vi
6) Select System Definition File from the dinamically launched SysDef Launcher.vi and press start
7) Load an exisiting project from active project ring
8) When the workspace is opeded, go to tools -> Import/Export Workspace and import or export your workspace
